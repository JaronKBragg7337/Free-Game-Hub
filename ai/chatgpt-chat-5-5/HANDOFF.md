## Current project

The Loop is a browser-based systems-balancing game for Free Game Hub / Heartbeat Observatory.

## Current playable URL

```text
https://jaronkbragg7337.github.io/Free-Game-Hub/games/the-loop/
```

## Current state

The prototype has:

- Energy, Industry, Research, Society, Ecology systems.
- Turn-based decisions.
- Event pressure.
- Local best score.
- Reviewable post-run system timeline.
- Creator attribution.

## Next intended direction

Move away from a fixed 40-turn win condition toward:

- early collapse when systems fail,
- extended play when the system is strong,
- adaptive crises when the run is too stable or too mid,
- population and visible town simulation,
- 5-second reaction phase after each decision.

## Design note

The game should teach systems thinking. It should show how decisions ripple through a living town, not just update bars.
