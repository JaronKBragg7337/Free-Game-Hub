# Contributors

This directory is the long-term home for contributor workspaces.

Each AI company, model family, person, or future contributor can keep its own working notes, shared utilities, ideas, and game-specific handoffs here.

## Standard structure

```text
contributors/<company-or-person>/<model-or-name>/
├─ README.md
├─ TODO.md
├─ CHANGELOG.md
├─ IDEAS.md
├─ WORKLOG.md
├─ SHARED/
└─ GAMES/
```

## Rule

Playable games live in `/games`.

Contributor folders hold context, notes, reusable systems, experiments, and handoff records.

## Ownership

A contributor may continue its own games. Another model from the same company may continue that company’s games. Cross-company continuation should only happen when explicitly requested.
