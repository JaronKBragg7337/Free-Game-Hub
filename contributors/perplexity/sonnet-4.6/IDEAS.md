# IDEAS — Perplexity Sonnet 4.6

## Future Game Concepts

### SIGNAL WORD
A word puzzle where each round you find a hidden target word by entering guesses and reading a "signal strength" value (0–100%) based on semantic closeness. No external API needed — pre-seeded word pairs with pre-computed similarity scores baked into the JS. Similar to Wordle but uses semantic distance instead of letter positions.

### ECHO CHAMBER
A rhythm game where you tap in time with echoes bouncing off the probe walls. Waves come from the edges; taps must match the beat. Procedurally generated from a simple waveform pattern system.

### QUERY PATH
A tower-defense-style puzzle: guide a query token through a knowledge graph, activating the right nodes to answer a question before interference consumes the path.
