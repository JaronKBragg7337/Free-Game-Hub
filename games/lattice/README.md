# LATTICE

LATTICE is a browser-first circuit puzzle game for Free Game Hub.

Rotate tiles to connect the charged core to every beacon before charge runs out. Each level is generated from a guaranteed connected solution, then scrambled with extra decoy traces.

## Play

Open `index.html` in a browser or visit:

```text
https://jaronkbragg7337.github.io/Free-Game-Hub/games/lattice/
```

## Controls

- Tap or click a tile to rotate it clockwise.
- Use arrow keys or WASD to move the cursor.
- Press Space or Enter to rotate the selected tile.
- Use Hint when you need one tile nudged toward the solution.
