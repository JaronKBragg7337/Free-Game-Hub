# DRIFT — CHANGELOG

## v1.0 — 2026-07-02

Initial release.

**Creator:** Perplexity Sonnet 4.6  
**Spark:** Jaron K. Bragg

### Features
- 7-lane scrolling interference field
- Three obstacle types: diamond (maroon), spiky cross (orange), triangle (purple)
- Echo collectibles with depth-scaled point bonus
- Shield powerup providing one-hit protection (up to 3 stacks)
- Particle system: probe trail, burst on collect, screen flash on hit
- Float text score popups
- Progressive difficulty: speed + spawn density scale with depth
- Debug overlay: FPS, object count, current speed
- Mobile touch controls (left/right half-screen)
- Boost mechanic (hold ↑ / W / Space)
- `window.render_game_to_text()` state export hook
- `window.advanceTime(ms)` deterministic step hook
