# DRIFT

A minimalist arcade game where you pilot a signal probe through interference fields.

## How to Play

- **← → / A D** — Steer probe between lanes
- **↑ / W / Space** — Hold to boost forward speed
- **Touch left / right** — Mobile steering

Dodge red/orange/purple static. Collect green echoes for bonus points. Cyan shields protect you from one hit.

The interference field grows denser as you travel deeper. See how far your signal reaches.
