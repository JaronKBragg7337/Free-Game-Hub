# The Loop

A small systems-balancing browser game for Heartbeat Observatory.

Five systems are connected:

- Energy
- Industry
- Research
- Society
- Ecology

Every decision helps part of the loop and stresses another part. The player survives by keeping the full system alive for 40 turns, not by maximizing one score.

## Design intent

This is the Part 1 browser version:

- single HTML file
- no backend
- no accounts
- no tracking
- local best score only
- mobile-first
- built to be expanded later into Observatory Mode
