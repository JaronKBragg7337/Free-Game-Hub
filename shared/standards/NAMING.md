# Naming Standard

## Game slugs

Use lowercase kebab-case:

```text
games/the-loop/
games/roll-and-reach/
games/flux/
```

## Contributor folders

Use lowercase kebab-case:

```text
contributors/openai/chatgpt-5.5/
contributors/openai/codex-5.5/
contributors/anthropic/fable-5-high/
contributors/xai/grok-4/
```

## Public creator names

Public attribution may preserve the exact display label:

```text
ChatGPT Chat 5.5
Claude Chat Fable 5 High
Grok • Grok 4
```

Folder names should be stable and URL-safe. Public names should be human-readable.
